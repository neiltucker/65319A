# Course 65319A: Advanced Automated Administration with Windows PowerShell

## Course Metadata

* **Course Number:** 65319A 
* **Duration:** 3 days 12 modules
* **Audience Level:** Advanced 
* **Format:** Instructor-Led / Bootcamp 

---

## Course Overview

This 3-day course teaches students how to automate administrative tasks using PowerShell on Windows Server 2025. Students will learn essential scripting skills and advanced automation techniques for Microsoft Azure, Active Directory, Entra ID, SQL Server 2025, virtual machines, remote administration, and file management. Special attention is focused on new capabilities introduced in Windows Server 2025 and PowerShell 7+.

### At Course Completion 

After completing this course, you will be able to:

* Implement advanced PowerShell scripting patterns and automation workflows for Windows Server 2025 environments.
* Automate identity and access management for Active Directory, Microsoft Entra ID, and hybrid scenarios.
* Deploy and manage Azure cloud infrastructure and on-premises virtualization platforms programmatically.
* Configure server infrastructure using Desired State Configuration and automate software deployment and patch management.
* Implement secure remote administration, file system management, and SQL Server 2025 database operations through automation.
* Automate network configuration, security monitoring, and compliance reporting for enterprise environments.

---

## Prerequisites

* Experience using PowerShell in a Server environment.
* Basic PowerShell scripting skills.
* Familiarity with Microsoft Azure cloud environment.
* Basic understanding of databases.

## Course Outline 

### Module 1: Advanced PowerShell Scripting Techniques
This module covers advanced scripting patterns, error handling, logging, and modern PowerShell features introduced in PowerShell 7+ and Windows Server 2025.
* **Duration:** 105 min 
* **Lab:** Build an Advanced PowerShell Module with Error Handling 

### Module 2: Identity Management with Active Directory
This module focuses on automating Active Directory administration tasks using PowerShell cmdlets for user, group, and organizational unit management.
* **Duration:** 105 min
* **Lab:** Automate Active Directory User and Group Management 

### Module 3: Microsoft Entra ID Automation and Management
This module covers automating Microsoft Entra ID (formerly Azure AD) administration including user management, group operations, and conditional access policies.
* **Duration:** 105 min 
* **Lab:** Automate Entra ID User and Access Management 

### Module 4: Azure Cloud Infrastructure Automation
This module focuses on automating Azure resource deployment and management using PowerShell Az modules and infrastructure-as-code principles.
* **Duration:** 105 min 
* **Lab:** Deploy and Manage Azure Infrastructure with PowerShell

### Module 5: Desired State Configuration and Server Management
This module covers implementing Desired State Configuration (DSC) for automated server configuration management and drift detection in Windows Server 2025.
* **Duration:** 105 min 
* **Lab:** Configure Server Infrastructure with DSC 

### Module 6: Hyper-V Virtualization Management with PowerShell
This module focuses on automating Hyper-V virtual machine management, including VM provisioning, snapshot management, and virtual networking configuration.
* **Duration:** 105 min 
* **Lab:** Automate Hyper-V Virtual Machine Management

### Module 7: Software Deployment and Patch Management Automation
This module covers automating software deployment, Windows Update management, and patch compliance reporting using PowerShell.
* **Duration:** 105 min 
* **Lab:** Implement Automated Software Deployment and Patching 

### Module 8: Remote Administration and Session Management
This module focuses on PowerShell remoting, remote session management, and secure remote administration techniques for distributed environments.
* **Duration:** 105 min 
* **Lab:** Configure and Implement Remote Administration 

### Module 9: File Systems, Storage, and Data Management
This module covers automating file system operations, storage management, and data lifecycle tasks using PowerShell on Windows Server 2025.
* **Duration:** 105 min
* **Lab:** Automate File System and Storage Operations

### Module 10: SQL Server 2025 Database Administration with PowerShell
This module focuses on automating SQL Server 2025 database administration tasks including backup operations, user management, and database maintenance.
* **Duration:** 105 min 
* **Lab:** Automate SQL Server 2025 Database Operations

### Module 11: Network Configuration and Management Automation
This module covers automating network configuration tasks including IP addressing, DNS management, firewall configuration, and network troubleshooting.
* **Duration:** 105 min 
* **Lab:** Automate Network Configuration and Security

### Module 12: Security Monitoring, Compliance, and Audit Automation
This module focuses on automating security monitoring, compliance reporting, and audit log analysis for Windows Server 2025 environments.
* **Duration:** 105 min 
* **Lab:** Implement Security Monitoring and Compliance Automation
